## Contributing
To contribute, simply **fork the repository**, edit the code, and **submit a pull request**.
We will then check your pull request and accept/deny it.
If you are clueless about what you could start with, [we collected some issues for you](https://github.com/flarialmc/launcher/contribute).
If you need any help with understanding our code, join the [official Discord server](https://flarial.xyz/discord)
